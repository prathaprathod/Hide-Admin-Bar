For WordPress Non Admins users Hides The Front-end Admin Bar.

== Description ==

Admin Bar Shows only for administrators not other users i mean non admins

When activated only administrators will see the admin bar.

This plugin has no settings and will not be updated often, so feel free to edit to your needs.

== Installation ==

1. Upload the `hide-admin-bar` directory to the `/wp-content/plugins/` directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. There are no settings, and this plugin will not be updated oftne.
